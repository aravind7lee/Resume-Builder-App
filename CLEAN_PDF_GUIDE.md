## Why This Happens

Browsers automatically add:
- **Top Left**: URL of the page
- **Top Right**: Date and time
- **Bottom Left**: Page title
- **Bottom Right**: Page numbers (1/2, 2/2, etc.)

This is a **browser feature**, not part of your resume!

## One-Time Setup

Good news! Once you disable "Headers and footers" in your browser, it usually **remembers this setting** for future downloads.

## Alternative Methods

### Method 1: Use Print Preview
1. Press `Ctrl+P` (Windows) or `Cmd+P` (Mac)
2. Disable headers/footers
3. Save as PDF

### Method 2: Browser Settings
Some browsers let you set default print preferences:
- Chrome: Settings â†’ Advanced â†’ Printing
- Firefox: about:config â†’ print settings

### Method 3: Use PDF Export Extensions
Install browser extensions that export clean PDFs without headers/footers.

## Pro Tips

1. **Save the setting**: Your browser remembers this preference
2. **Use keyboard shortcut**: `Ctrl+P` or `Cmd+P` for quick access
3. **Check preview**: Always preview before saving
4. **Test once**: Download one resume to verify settings

## Still Seeing Headers/Footers?

If you still see them after unchecking:
1. Try a different browser
2. Clear browser cache
3. Update your browser to latest version
4. Use "Save as PDF" instead of "Print to PDF"
